HTML标签完整参考表格

包含基础标签与扩展标签，每个标签都有详细说明和代码示例

基础文本与结构标签

标签 中文意思 英文原意 注意事项 代码示例
<p> 段落 Paragraph 块级元素，浏览器自动添加上下边距 <p>这是一个段落</p>
<div> 分区/容器 Division 块级元素，用于布局和样式化，无语义 <div class="container">内容</div>
<span> 内联容器 Span 内联元素，用于文本部分样式化，无语义 <span class="highlight">重要</span>
<br> 换行 Break 空元素，在XHTML中必须自闭合<br /> 第一行<br>第二行
<hr> 水平线 Horizontal Rule 空元素，表示主题内容的变化 <hr>
<a> 超链接 Anchor 必须使用href属性，target="_blank"新窗口打开 <a href="page.html">链接</a>
<img> 图像 Image 空元素，必须使用src和alt属性 <img src="pic.jpg" alt="图片">
<ul> <ol> <li> 无序/有序列表 Unordered/Ordered List, List Item ul无序列表，ol有序列表，li列表项 <ul><li>项目</li></ul>
<table> <tr> <td> 表格/行/单元格 Table, Table Row, Table Data table定义表格，tr定义行，td定义单元格 <table><tr><td>数据</td></tr></table>
<b> 粗体 Bold 仅表示视觉上的粗体，无强调语义 <b>粗体文本</b>
<em> 强调 Emphasis 表示语义上的强调，通常显示为斜体 <em>强调文本</em>
<mark> 标记 Mark 表示突出显示或标记的文本 <mark>标记文本</mark>
<del> 删除 Delete 表示已删除的文本，通常有删除线 <del>删除文本</del>

语义化结构标签

标签 中文意思 英文原意 注意事项 代码示例
<header> 页眉/头部 Header 表示介绍性内容或导航链接的容器 <header><h1>标题</h1></header>
<main> 主要内容 Main 一个页面只能有一个，表示页面的主要内容 <main>主要内容</main>
<section> 章节/部分 Section 表示文档中的一个独立部分，通常有标题 <section><h2>章节</h2></section>
<nav> 导航 Navigation 表示页面导航部分，通常包含链接列表 <nav><a href="#">链接</a></nav>
<aside> 侧边栏/旁注 Aside 表示与主要内容相关但不是主要内容的部分 <aside>相关内容</aside>
<footer> 页脚 Footer 表示页面或章节的页脚 <footer>版权信息</footer>
<article> 文章 Article 表示独立、完整、可独立分发的内容 <article><h2>文章标题</h2></article>

表单与输入标签

标签 中文意思 英文原意 注意事项 代码示例
<form> 表单 Form 用于创建包含输入控件的表单 <form action="/submit" method="post"></form>
<input> 输入框 Input 空元素，type属性决定输入类型 <input type="text" name="user">
<label> 标签 Label 为表单元素定义标签，使用for属性关联 <label for="name">姓名</label>
<select> <option> 下拉列表/选项 Select / Option select创建下拉列表，option定义选项 <select><option>选项</option></select>
<fieldset> <legend> 字段组/图例 Fieldset / Legend fieldset分组表单元素，legend定义标题 <fieldset><legend>组</legend></fieldset>
<datalist> 数据列表 Datalist 为输入框提供预定义选项列表 <input list="options"><datalist id="options"></datalist>
<textarea> 文本区域 Textarea 创建多行文本输入框 <textarea rows="4" cols="50"></textarea>
<button> 按钮 Button 创建可点击的按钮 <button type="submit">提交</button>

媒体与交互标签

标签 中文意思 英文原意 注意事项 代码示例
<video> 视频 Video 用于嵌入视频内容，支持多种属性 <video controls><source src="video.mp4"></video>
<audio> 音频 Audio 用于嵌入音频内容，支持多种属性 <audio controls><source src="audio.mp3"></audio>
<canvas> 画布 Canvas 用于通过JavaScript动态绘制图形 <canvas id="myCanvas"></canvas>
<details> <summary> 详情/摘要 Details / Summary 创建可折叠的内容区域 <details><summary>标题</summary>内容</details>
<progress> 进度条 Progress 表示任务的完成进度 <progress value="70" max="100"></progress>
<figure> <figcaption> 图表/图表标题 Figure / Figure Caption figure表示独立内容，figcaption定义标题 <figure><img src="img.jpg"><figcaption>标题</figcaption></figure>
<meter> 度量 Meter 表示已知范围内的标量测量值 <meter value="0.6">60%</meter>
<output> 输出 Output 用于显示计算或用户操作的结果 <output name="result">0</output>

文档与元数据标签

标签 中文意思 英文原意 注意事项 代码示例
<html> HTML文档 HTML Document 表示HTML文档的根元素 <html lang="zh-CN"></html>
<head> 头部 Head 包含文档的元数据和链接信息 <head><title>标题</title></head>
<body> 主体 Body 包含文档的所有可见内容 <body>页面内容</body>
<title> 标题 Title 定义文档标题，显示在浏览器标签页上 <title>页面标题</title>
<meta> 元数据 Metadata 空元素，提供关于HTML文档的元数据 <meta charset="UTF-8">
<link> 链接 Link 空元素，用于链接外部资源如CSS文件 <link rel="stylesheet" href="style.css">
<style> 样式 Style 用于在文档中嵌入CSS样式 <style>body { color: red; }</style>
<script> 脚本 Script 用于嵌入或引用JavaScript代码 <script>console.log('Hello')</script>
