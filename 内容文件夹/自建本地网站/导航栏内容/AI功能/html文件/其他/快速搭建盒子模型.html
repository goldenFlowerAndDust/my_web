脑补训练（闭眼想象）

  1. header → 放 Logo + 导航菜单
  2. hero → 放大标题 + 主按钮 + 背景图
  3. main → 放文章卡片 / 商品列表 / 表单
  4. sidebar → 放分类、广告、热门标签
  5. footer → 放版权、备案号、社交链接

→ 以后任何网页，先画 5 个框，再填内容，再细化样式。

  • ☐ 复制上面 5 段 <section> → 改 class 名 → 填真内容
  • ☐ 换背景图：.hero{background:url(图片) center/cover}
  • ☐ 换颜色：改 CSS 变量 --main
  • ☐ 手机测试：缩小浏览器，确认竖排正常
  • ☐ 上线：push → GitHub Pages → 自定义域名
