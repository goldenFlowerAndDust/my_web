网页制作全流程（零→上线）

照着顺序做，一步不落；先搭盒子，后填内容，最后细化样式 → 部署。

① 画草图（30 秒手绘）

任何网页都能拆成 5 个盒子：

  1. 头部（Header）—— Logo + 导航
  2. 主视觉（Hero）—— 大标题 + 按钮
  3. 内容区（Main）—— 列表/表单/文章
  4. 侧边栏（Aside）—— 广告/分类
  5. 底部（Footer）—— 版权/链接

→ 画完草图再写代码，不会乱。

② 搭盒子（HTML 骨架）

只写结构，不加样式；先让盒子竖排能看。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>我的网页</title>
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>
<body>
  <!-- 头部 -->
  <header>
    <div class="logo">Logo</div>
    <nav>导航</nav>
  </header>

  <!-- 主视觉 -->
  <section class="hero">
    <h1>大标题</h1>
    <p>一句话卖点</p>
    <a href="#" class="btn">立即体验</a>
  </section>

  <!-- 内容区 -->
  <main>
    <h2>最新文章</h2>
    <ul>
      <li>文章 1</li>
      <li>文章 2</li>
    </ul>
  </main>

  <!-- 侧边栏 -->
  <aside>
    <h3>分类</h3>
    <ul>
      <li>分类 A</li>
      <li>分类 B</li>
    </ul>
  </aside>

  <!-- 底部 -->
  <footer>
    © 2025 我的网站
  </footer>
</body>
</html>

→ 保存后浏览器打开，看到 5 块竖排文字 = 成功。

③ 填内容（替换文本/图片）

  • Logo → 放 <img src="images/logo.png">
  • 导航 → 放 ul/li 链接
  • 文章 → 放 <article> 标题+摘要+日期
  • 图片 → 先建 images/ 文件夹，统一放图

→ 内容填完再美化，避免“空盒子调样式”。

④ 加样式（CSS 顺序）

  1. 全局：重置、字体、颜色变量
  2. 布局：.container、.row、.col（Flex/Grid）
  3. 组件：按钮、卡片、表单
  4. 细节：圆角、阴影、过渡
/* 全局变量 */
:root{ --main:#2196f3; --gap:20px; }

/* 布局：先让盒子横排 */
.row{display:flex;flex-wrap:wrap;margin:calc(var(--gap) / -2);}
.col{flex:1 1 300px;margin:calc(var(--gap) / 2);}

/* 组件：按钮 */
.btn{display:inline-block;padding:10px 20px;background:var(--main);color:#fff;border-radius:4px;}

→ 每写完一条就刷新，确认“盒子没散”再往下写。

⑤ 响应式（媒体查询）

@media (max-width: 768px){
  .row{flex-direction:column;} /* 手机竖排 */
  .col{flex:1 1 100%;}
}

→ 先写桌面，再写 768px 断点；只改布局，不改内容。

⑥ 上线（零成本）

  1. 本地 git init → 推送到 GitHub
  2. 仓库 Settings → Pages → 选 main 分支 → Save
  3. 得到 https://你的用户名.github.io/仓库名
  4. （可选）Freenom 免费域名 + DNS 指向 → 自定义网址
打开 GitHub Pages 设置

⑦ 检查清单（打印出来打钩）

  • ☐ 草图：5 个盒子画完
  • ☐ HTML：骨架无样式能看
  • ☐ 内容：Logo、文章、图片真材实料
  • ☐ CSS：先全局→布局→组件→细节
  • ☐ 响应：768px 断点测试通过
  • ☐ 上线：GitHub Pages 能打开
  • ☐ 自定义：免费域名绑定成功

全部打钩 = 网站上线，随时可分享！
