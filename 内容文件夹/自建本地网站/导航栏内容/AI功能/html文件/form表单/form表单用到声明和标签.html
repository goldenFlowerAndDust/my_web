HTML表单标签与属性参考表

移动端优化版 - 包含所有常用表单标签和属性

表单结构标签

标签 中文意思 说明与示例
<form> 表单容器 定义包含表单元素的HTML表单
<form action="/submit" method="post"></form>
<fieldset> 字段组 将相关表单元素分组
<fieldset>...表单元素...</fieldset>
<legend> 组标题 为fieldset元素定义标题
<legend>个人信息</legend>
<label> 标签 为表单元素定义标签，使用for属性关联
<label for="name">姓名:</label>

输入控件标签

标签 中文意思 说明与示例
<input> 输入框 创建各种类型的输入控件，type属性决定类型
<input type="text" name="user">
<textarea> 文本区域 创建多行文本输入框
<textarea rows="4" cols="50"></textarea>
<select> 下拉列表 创建下拉选择列表
<select name="country">...</select>
<option> 选项 定义select元素中的选项
<option value="1">选项1</option>
<optgroup> 选项组 将相关选项分组
<optgroup label="组1">...</optgroup>
<datalist> 数据列表 为输入框提供预定义选项
<datalist id="list">...</datalist>
<button> 按钮 创建可点击的按钮
<button type="submit">提交</button>
<output> 输出 显示计算或用户操作的结果
<output name="result">0</output>

常用表单属性

属性 适用元素 说明与示例
name 所有表单控件 定义表单元素的名称，用于数据提交
<input name="username">
value input, button, option 定义元素的初始值
<input value="默认值">
placeholder input, textarea 提供输入提示文本
<input placeholder="请输入...">
required input, select, textarea 指定字段为必填项
<input required>
disabled 所有表单控件 禁用表单元素
<input disabled>
readonly input, textarea 设置字段为只读
<input readonly>
maxlength input, textarea 限制输入的最大字符数
<input maxlength="10">
min / max input(number/date/range) 设置数值/日期范围
<input min="1" max="100">
pattern input(text/tel/email/url) 指定输入值的正则表达式验证
<input pattern="[A-Za-z]{3}">
autocomplete form, input 启用/禁用自动完成功能
<input autocomplete="off">
autofocus 所有表单控件 页面加载时自动聚焦到元素
<input autofocus>
multiple input(file/email), select 允许多个值/文件
<input type="file" multiple>
form 所有表单控件 指定元素所属的表单
<input form="myForm">
list input 引用datalist元素
<input list="options">
selected option 预设选中选项
<option selected>默认</option>
checked input(checkbox/radio) 预设选中状态
<input type="checkbox" checked>
for label 指定label关联的表单元素
<label for="inputId">标签</label>

form标签属性

属性 说明 示例
action 指定表单提交的URL
<form action="/submit">
method 指定HTTP方法(get/post)
<form method="post">
enctype 指定表单数据的编码类型
<form enctype="multipart/form-data">
target 指定响应显示的位置
<form target="_blank">
novalidate 禁用表单验证
<form novalidate>
