样式和标签总结

标签 属性 说明
html lang="en" 设置文档的语言为英语
head 无 包含文档的元数据
meta charset="UTF-8" 设置字符编码为UTF-8
title 无 设置文档的标题
style 无 定义文档的样式
body 无 包含文档的主体内容
main 无 定义文档的主要内容区域
form 无 定义表单
fieldset 无 定义表单中的字段集
legend 无 定义字段集的标题
div 无 定义文档的一个分区或节
label for="id" 定义表单控件的标签，`for` 属性与表单控件的 `id` 一致
input type="text" 定义单行文本输入框
input type="password" 定义密码输入框
input type="tel" 定义电话号码输入框
input type="email" 定义电子邮件地址输入框
input type="checkbox" 定义复选框
input type="radio" 定义单选按钮
input type="submit" 定义提交按钮
input type="reset" 定义重置按钮
select 无 定义下拉列表
optgroup label="text" 定义下拉列表中的选项组
option 无 定义下拉列表中的选项
@keyframes gradient 定义动画的关键帧
animation gradient 15s ease infinite 应用动画，`gradient` 是动画名称，`15s` 是动画时长，`ease` 是动画速度曲线，`infinite` 表示无限循环
background linear-gradient(270deg, #74ebd5, #acb6e5, #f1f8e9) 定义线性渐变背景
background-size 600% 600% 定义背景的大小
background-position 0% 50% 定义背景的位置
display flex 定义弹性布局
justify-content center 在弹性布局中，水平居中对齐子元素
align-items center 在弹性布局中，垂直居中对齐子元素
padding 10px 15px 定义内边距
margin 10px 0 定义外边距
cursor :pointer 定义鼠标指针的形状，`pointer` 表示鼠标指针变为手形
outline none 移除元素的轮廓
:hover background-color: #f0f0f0; 定义鼠标悬停时的背景颜色
