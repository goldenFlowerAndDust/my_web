标签——水平线

  • 解释
    1. 水平线标签(hr)
    2. 该标签是在HTML页面中创建水平线
  • 定义CSS样式属性
    1. color:设置水平线的颜色
    2. width:设置水平线的宽度
    3. height:设置水平线的高度
    4. align:设置水平线的对其方式(默认居中)，可取值left(左)，right(右)

未使用水平线标签

使用水平线标签并定义CSS样式属性

    内部调用

    1. CSS属性样式，都在head标签内，新建的style标签内部
    2. 因为水平线不属于文本，单独使用color属性，没有效果，需要border搭配使用
      则：border-color:red
    3. width和height不受影响可以直接用
    4. 在HTML5里面hr标签的align被废除了，也就是说，无法直接用center、left、right直接定义对齐方式了
      被margin代替：

      5. 具体的形式是：
      靠左：margin-left：数值(单位如：px……)
      靠右：margin-right：数值(单位如：px……)
      居中：margin-left：auto 以及 margin-right：auto

      名词解释——auto

      auto就是告诉浏览器，帮我算的意思:

    5. margin:auto家族——块级水平对齐专用

        6. 属性：

      • margin-left
      • margin-right
      • margin-top
      • margin-boottom
        • 取值:auto

        计算规则：把可用剩余空间全部吸收到自己身上

      • 行内元素→0(无效)

      • 块级元素宽度非auto时:

        • 当left与right同时为auto时，剩余空间平分，最后的结果是水平线居中
        当left与right仅一侧为auto，该恻吃掉全部剩余空间，最后的结果是靠左或靠右

      • 块级元素高度非auto且父级高度确定时:

        • 上下同时为auto→0(CSS2.1规则)
        Flex容器里再设置margin-top/bottom:auto→垂直真•居中或靠底
