

# 字体属性—背景属性—文本属性——表格属性

## 字体属性

定义：CSS字体属性定义字体:颜色、大小、加粗、文字样式

### 颜色

定义：规定文本颜色
  一共四种方法：

- 英文意思，如：red、yellow、blue、black
- 十六进制(#开头，接0\~9与A\~F整数区间的数，共六位)，如：黑(#000000)
- rgb(颜色通道定义)\[共需输入三个数值，每个数值的范围在0\~255区间\]，如:黑\[rgb(0,0,0)\]
- rgba(颜色通道定义)\[共需输入四个数值，前三个与rgb一样，第四个数值0\~1,意思就是颜色的深浅度，0完全透明，1完全不透明\]，如：黑色_半透明\[rgba(0,0,0,0.5)\]

### font-size

- 定义：设置文本大小
- 用意：管理文字的大小，在网页设计中是非常重要的。
- 注意:
  1.  但是，不能通过调整字体大小使段落看上去像标题，或则使标题看上去像段落
  2.  不同浏览器和设备可能有不同的最小字体限制，例如，Chrome浏览器接受的最小字体大小通常是12px，但具体值可能因设备和用户设置而异
- 示例：p{font-size:25px}

### font-weight

- 定义：设置文本的粗细
- 属性值
  1.  bold：定义粗体字符

  2.  bolder：定义更粗的字符

      注意：bold与bolder在当今浏览器显示效果是一样的，所以一般用值：bold就可以了

  3.  lighter:定义更细的字符

  4.  100\~900：定义由细到粗，400等同于默认，700等同于bold

### font-style

- 定义:指定文本的字体样式
- 属性值：
  1.  normal：默认值
  2.  italic:定义斜体字
- 注意：设置的样式不仅可以将非斜体字改为斜体字，还可以将I标签(默认是斜体字)，改为默认字体

### font-family

- 定义：font-family属性指定一个元素的字体
- 温馨提示：
  1.  可以添加多个字体(值)，每个值用逗号隔开
  2.  如果字体名称包含空格，它必须加上引号
  3.  字体使用方式:由前到后，前面失效或无法使用，后面排上去
- 如：p{font-family:\"微软雅黑\",\"华文宋体\",serif;}

## 背景属性

### background-color属性

- 定义:设置背景颜色


```text
.box{
	background-color:red;（颜色同样有四种方式）
}
```

### background-image属性

- 定义：设置元素的背景图像

- 解释：元素的背景是元素的总大小，包括填充和边界(不包括外边距)。默认情况下background-image属性放置在元素的左上角，如果图像不够大的话会在垂直方向和水平方向平铺图像，如果图像大小超过元素大小从图像的左上角显示元素大小那部分。

- 具体过程：
  1.  先在body标签内，建立div标签，并引用对应的选择器(可以是元素选择器、类选择器、ID选择器等)

  2.  在style标签内，对应的选择器添加样式，设置高、宽，以及最主要的：background-image属性声明

  3.  声明语法：background-image:url(\"图片具体地址\")

  4.  代码演示：


```text
/* 可以先设置一个div容器，定义宽度和高度，将图片装进这个容器里面，图片的显示部分以容器大小决定。*/
.div_1{
	width:500px;
	height:500px;
	background-image:url("图片具体地址");
}
```
- 5. 显示效果：

     - 整个过程意思就是：设置一个div容器，定义宽度和高度，将图片装进这个容器里面，图片的显示部分以容器大小决定。

     - 图片原本样子：

     - 容器内的样子：
       ![使用声明，给容器添加背景](素材存放区/背景属性/使用background-image声明，给容器添加背景.png)

### background-repeat属性(设置图片平铺方式)

- 定义：该属性设置如何平铺背景图像
  - 解释：
  - 该声明就是，当设置的容器过大，且图片过小时，该怎么显示背景图片
  - 默认水平、垂直同时平铺(就是重复显示)
    如：![background-repeat------默认](素材存放区/背景属性/使用background-repeat声明——默认.png)
- 声明的值及对应解释:
    ----------- ------------------
    |    值     |       说明       |
    | :-------: | :--------------: |
    |  repeat   |      默认值      |
    | repeat-x  | 只向水平方向平铺 |
    | repeat-y  | 只向垂直方向平铺 |
    | no-repeat |      不平铺      |

----------- ------------------

  - 各个值的平铺方式演示：
  - 默认图片：![默认](素材存放区/背景属性/使用background-repeat声明——默认.png)
  - 水平：![水平](素材存放区/背景属性/使用background-repeat声明——水平.png)
  - 垂直：![垂直](素材存放区/背景属性/使用background-repeat声明——垂直.png)
  - 不平铺:![不平铺](素材存放区/背景属性/使用background-repeat声明——不平铺.png)
- 总结：
  - background-repeat指的就是容器过大时背景图片的显示方式
  - repeat-x：表示背景图像只在水平方向上重复平铺
  - repeat-y:表示背景图像只在垂直方向上重复平铺
  - no-repeat:表示背景图像不会重复平铺，只显示一次
  - 注意：no-repeat是，背景图像只显示一次，不会重复填充整个容器。容器内的空间仍然存在，只是背景图像不会重复显示。

### background-size属性：

- 定义：该属性设置背景图像的大小
- 各个值及说明

    | 值         | 说明                                                         |
    | ---------- | ------------------------------------------------------------ |

    | percentage | 计算相对位置区域的百分比，第一个值为宽度，第二个值为高度，如果只设置一个值，第二个值为auto |
    | cover      | 保持图片纵横比并将图片缩放成完全覆盖背景区域的最小大小       |
    | contain    | 保持图片纵横比并将图像缩放成适合背景定位区域的最大大小       |
    
    ------------ --------------------------------------------------------------------------------------------
    
    - 各个值，演示

        ![length演示](素材存放区/背景属性/使用background-size_length声明演示.png)

        ![percentage演示](素材存放区/背景属性/使用background-size_percentage声明演示.png)
      - 使用cover设置图片大小(浏览器会将图片缩放到最小尺寸，同时保持宽高比例，确保图片完全覆盖容器的整个区域。如果容器的宽高比与图片的宽高比不同，图片可能会被裁剪，以适应容器的尺寸。这与背景图像是否平铺无关，通常background-repeat:no-repeat使用，以确保背景图像只显示一次\])
        ![cover演示](素材存放区/背景属性/使用background-size_cover演示.png)
      - 使用contain设置图片大小(contain为属性值\[浏览器会将图片缩放到最大尺寸，同时保持宽高比例，确保图片完全显示在容器内。如果容器的宽高比与图片的宽高比不同，图片可能会在容器内留白。结合使用background-repeat: no-repeat，可以确保背景图像只显示一次，不会重复平铺。\])

## 表格属性

### 表格边框(border属性)

- 定义：指定CSS表格边框，使用border属性

- 代码示例：


    table,td{
    	border:1px solid black;
    }

其中:
       border为属性声明，第一个值为边框的宽度(越高越宽)
       第二个值为边框的形式(solid\[实线\],dashed\[虚线\],dotted\[点线\],double\[双实线\],none\[无边框\])
       第三个值为颜色(同样适用\[英文、十六进制数、rgb、rgba\])
